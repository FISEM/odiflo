# Odiflo

Public monorepo for Odiflo open source packages.

## Packages

- [@odiflo/easy-intl](./packages/easy-intl) - Component-level i18n with formatters

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Testing

```bash
pnpm test
```

## Building

```bash
pnpm build
```
